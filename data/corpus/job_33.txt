**Responsibilities**
* Thought leader and owner of the architecture blueprint of data platform
* Superior coding skills in SPARK using Scala, Python and Java
* Work collaboratively with other Platform, infrastructure, application engineers as well as AI Researchers to build next generation data platform services and capabilities.

**Qualifications**
**Required Qualifications:**
* Bachelor's Degree in Computer Science, Math, Software Engineering, Computer Engineering, or related field AND 8\+ years experience in software development, data modeling or data engineering work
* OR Master's Degree in Computer Science, Math, Software Engineering, Computer Engineering, or related field AND 6\+ year(s) experience in software development, or data engineering work
* OR equivalent experience.
* 5\+ years of hands experience in building from ground up large scale data processing platforms with more than 100 data pipelines. Use of Apache Hadoop eco system, Kafka Stream/Flink, columnar databases, vector databases, semantic search technologies, RDBMS databases, key value stores, etc.
* 5\+ years' experience building scalable services on top of public cloud infrastructure like Azure, AWS, or GCP.
* 2\+ years' experience building distributed systems at scale and extensive systems knowledge that spans bare\-metal hosts to containers to networking.


**Preferred Qualifications**
* 10\+ years technical engineering experience building data platforms that handle large volumes of data in both batch and real time. Coding in languages including, but not limited to, Python, C\#, Java, Spark, SQL
